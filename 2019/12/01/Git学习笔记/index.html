Git简介

Git是什么

Git是一个分布式的版本控制系统，由Linux之父Linuz为了更好的管理Linux内核开发而设计。

Git的特点

  • Git是一个免费且开源的软件。

  • 和CVS和SVN等集中式版本控制软件不同，Git是一个分布式的版本控制系统，可以不依赖于中央主机和网络而独立运行。

安装Git

在Linux上安装Git

在提供软件包管理器的Linux上可以使用包管理器一键安装。

Ubuntu等Debian系Linux：sudo apt install git

Archlinux系Linux：sudo pacman -S git

若没有包管理器或者不想用包管理器安装的，可以选择编译安装。

在Windows上安装Git

可以在https://desktop.github.com/上下载Github for Windows并且根据提示安装。

也可以在cmder官网上下载cmder的Full版本，解压后开箱即用，并且自带Git。

远程仓库

虽然Git的使用不需要网络，但在多人合作时，一般还是得需要一台24小时工作的服务器来帮助管理项目。

使用者从服务器里克隆项目，做出修改后再提交到服务器，服务器上的仓库便称为远程仓库。

使用Github

Github是一个提供Git仓库托管服务的网站，对公有仓库是免费的。

首先创建本地机器的SSH Key：

一路回车，这会在用户家目录生成一个.ssh目录，里面有id_rsa和id_rsa.pub两个文件，其中id_rsa为私钥，不能公开，id_rsa.pub为公钥，可以公开，复制id_rsa.pub文件的内容，打开GitHub，点开个人设置，点击SSH and GPG keys，再点New SSH Key按钮。

Title可以随便填，Key填id_rsa.pub文件的内容，点Add SSH Key，然后你本地机器的SSH Key就添加完成了。
