Git简介

Git是什么

Git是一个分布式的版本控制系统，是由Linux之父Linuz为了更好的管理Linux内核开发而设计。

Git的特点

  • Git是一个免费且开源的软件。

  • 和CVS和SVN等集中式版本控制软件不同，Git是一个分布式的版本控制系统，可以不依赖于中央主机和网络而独立运行。

安装Git

在Linux上安装Git

在提供软件包管理器的Linux上可以使用软件包管理器一键安装。

Ubuntu等Debian系Linux：sudo apt install git

Archlinux系Linux：sudo pacman -S git

若没有包管理器或者不想用包管理器安装的，可以选择编译安装。
